*MI PRIMER CRUD EN JAVA*

Este proyecto consiste en un sistema de gestión (CRUD) desarrollado con una arquitectura de capas, diseñado para interactuar de forma eficiente con una base de datos relacional.
Se utilizó Java para construir la lógica de negocio y la estructura de objetos. Gracias a su enfoque en la Programación Orientada a Objetos (POO), el sistema es escalable y organizado en paquetes (Lógica, Persistencia e IGU).


*Persistencia de Datos: JPA (Java Persistence API)*
Actúa como el puente inteligente entre el código Java y la base de datos, se utilizó un proveedor de persistencia EclipseLink.

*Entorno de Servidor: WAMP Server*
WAMP proporciona la infraestructura local necesaria para que el sistema funcione en Window

*Gestión de Base de Datos: phpMyAdmin*
Es la herramienta gráfica basada en web que permite administrar la base de datos MySQL de forma sencilla.
